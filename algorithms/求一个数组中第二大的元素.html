# 求一个数组中第二大的元素

要求：

  • 不能对这个数组进行整体排序；
  • 若要用循环，只能一重循环；
  • 不适用额外空间。
var arr = [1, 3, 5, 2, 7, 6]
arr[arr.indexOf(Math.max.apply(null, arr))] = null
Math.max.apply(null, arr)
